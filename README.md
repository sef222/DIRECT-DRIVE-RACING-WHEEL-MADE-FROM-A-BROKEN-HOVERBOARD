# DIRECT-DRIVE-RACING-WHEEL-MADE-FROM-A-BROKEN-HOVERBOARD

A DIY **direct-drive force feedback racing wheel** for PC built using a broken hoverboard motor, open-source firmware, and off-the-shelf electronics.

> ⚠️ This project is currently under development.

## Features

- 🛞 Direct-drive force feedback
- 🖥️ PC compatible
- ⚙️ Hoverboard hub motor
- 🎮 Custom-built 3-pedal set
- 📊 Custom telemetry dashboard
- 🔧 Open-source hardware and firmware
- 🧩 Modular design for easy upgrades

## Hardware

- Hoverboard hub motor
- MKS XDrive Mini / ODESC
- STM32F407VET6
- AS5047P magnetic encoder
- 36V power supply
- Custom aluminum frame
- DIY steering wheel
- DIY load-cell pedals
- Emergency stop switch

## Goals

- Match or exceed entry-level direct-drive wheels.
- Keep the project affordable.
- Make everything as open and documented as possible.
- Design a clean enclosure that can be CNC machined or 3D printed.
- Add features usually found only on expensive commercial wheels.
